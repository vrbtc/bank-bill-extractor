## 📁 文件说明

### 核心文件

#### this_month_bills.py（主程序）
- **作用**：协调器，调用各模块完成账单查询
- **改动**：从671行精简到276行
- **依赖**：email_client, email_decoder, bank_extractors

#### bank_extractors.py（银行提取器）
- **作用**：包含7个银行的专用提取逻辑
- **关键类**：
  - CCBBankExtractor → 建设银行（✅已修复）
  - CMBCBankExtractor → 民生银行（✅已修复）
  - CommBankExtractor → 交通银行
  - SPDBankExtractor → 浦发银行
  - PSBCBankExtractor → 邮储银行
  - CMBBankExtractor → 招商银行
  - GuangfaBankExtractor → 广发银行

#### email_decoder.py（邮件解码）
- **作用**：处理MIME编码和HTML解码
- **方法**：
  - decode_mime_words() - 解码邮件主题
  - decode_html_content() - 解码HTML内容

#### email_client.py（邮箱客户端）
- **作用**：管理IMAP邮箱连接
- **功能**：connect(), fetch_emails(), disconnect()

#### html_parser.py（HTML解析器）
- **作用**：清理HTML并转换为Markdown
- **内部使用**：被bank_extractors调用
